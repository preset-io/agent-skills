# Preset MCP Skills

- no `.codex-plugin/plugin.json`

MCP skill design and packaging will be handled in a separate task or PR.

This placeholder should only be promoted to an installable plugin after the MCP workflow boundaries, skill routing rules, and client manifests are designed together.
